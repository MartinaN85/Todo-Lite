# Todo Lite

Ett enkelt och responsivt todo-verktyg byggt med HTML, CSS och JavaScript.  
Stöd för att lägga till, redigera, markera och ta bort todos – med visuell bekräftelse och räknare.

---

# Funktioner

- Lägg till nya todos
- Redigera befintliga todos
- Markera som klar eller aktiv
- Ta bort med bekräftelseruta och overlay
- Räknare för aktiva todos
- Enhetlig styling för knappar, inputfält och dialoger

---

# Tekniker

- HTML5, CSS3 (flex/grid)
- JavaScript (DOM-manipulation, eventhantering)
- Git & GitHub (PR-flöde, branching)
- Trello (user stories, estimering med VP/SP/BFB)
